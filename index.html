The Web

The web is a collections of web documents made with HTML language connected through hyperlinks. It consists of user, his computer w/browser, Internet and Servers How it works? You post a request that goes through your browser via Internet to servers (through quite simple http ptotocol), and the server sends back files

HTML

HTML docs are the hear of the web. They consist of text content, markup, references to other docs, link to thoer pages. Markup is made of tags.

Tags

Tags can be of various types - mostly they have and opening and closing tag (Opening Tag+Content+Closing Tag=Element), though there can be void tags like 'br' which don't have a clisng one. There are also inline and block elements. I also got familiar with the general structure of the HTML document, which you can see implemented here.
